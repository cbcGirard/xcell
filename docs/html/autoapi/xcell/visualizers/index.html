xcell.visualizers._add_inset(baseAxis, rInset, xmax, relativeLoc=(0.5, -0.65))#

Create sub-axis for plotting a zoomed-in view of the main axis.

Plot commands executed in the main axis DO NOT automatically appear in inset; to keep them synchronized, use a pattern like

axes=[mainAx,inset] for ax in axes:

ax.plot_command(…)

Parameters:

  • baseAxis (axis) – Parent axis.

  • rInset (float) –

    Size of inset bounds (such that bounding box is

    rInset*[-1,1,-1,1])

  • xmax (float) – Size of parent axis’ bounds.

  • relativeLoc (TYPE, optional) – Placement of inset’s center, relative to parent axis. The default is (.5, -.8), which plots directly beneath the main plot.

Returns:

inset – Inset axis.

axis
