Add a biphasic pulse to the signal.

Parameters:

  • t_start (float) – Time of pulse initiation

  • pulse_duration (float) – Duration of a pulse phase in seconds.

  • pulse_amplitude (float) – Amplitude of pulse. Use a negative value to place the negative phase first.

  • interphase (float) – Time between phases of the pulse (default 0.) Not yet implemented.
